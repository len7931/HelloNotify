HelloNotify
===========

Another Notification app for android, which can control wifi, mobile data, screenbrightness etc.



需求分析

有些手机没有toolbox，鉴于此问题，开发一个notification 应用，完成类似功能。
需求如下：

1. 在手机下拉菜单增加一个常驻通知栏(Hello Notification)
2. 常驻通知栏(Hello Notification) 包含如下几个按钮：
	wifi 按钮： 	控制wifi 开关，点击此按钮反转当前wifi 的状态。（开――》关，或者 关――》开）
	2G 按钮：	控制2G/3G 数据业务开关，点击此按钮反转当前手机数据业务的状态。
	+ 按钮：		增加当前屏幕亮度，点击此按钮增加当前手机屏幕亮度。
	- 按钮：		降低当前屏幕亮度，点击此按钮降低当前手机屏幕亮度。
